Tópico 2: Manipulação do DOM (Página 30)

Exemplo Real: Um exemplo real de manipulação do DOM é a criação de uma lista de tarefas interativa. Usando JavaScript, os desenvolvedores podem permitir que os usuários adicionem, removam e marquem tarefas como concluídas. Aqui está um exemplo de código JavaScript para adicionar uma nova tarefa a uma lista:

  • Estudar
  • Pesquisar
  • Ler
